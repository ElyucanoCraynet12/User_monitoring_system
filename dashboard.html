    Quick operational tips to keep your system healthy:

    • Regularly monitor temperature, pH, and oxygen levels.
    • Perform partial water changes weekly.
    • Remove uneaten food to prevent ammonia spikes.

      User Manual

      Overview

      This manual helps you install, wire, and operate the ELYUCANO Crayfish Monitoring & Control system. It matches the look and accessibility of the dashboard and guides you through common issues.

      Key components

      • Sensors: Temperature, pH, Dissolved Oxygen, Turbidity.
      • Actuators: Feeder, Water Drain, Water Refill (controlled via relays).
      • Connectivity: ESP32 with Wi‑Fi and Firebase logging.

      System Setup (Software)

      1. Install the Arduino IDE (v2.x recommended). Add ESP32 boards via Boards Manager. Select ESP32 Dev Module and the correct COM port.
      2. Install required sensor libraries. If present, follow ARDUINO_LIBRARY_SETUP.md.
      3. Open the firmware file (for example ESP32_Smart_Water_Monitoring_System.ino or crayfish_monitoring_system.ino).
      4. Update Wi‑Fi SSID and password and ensure Firebase keys match those used by dashboard.html.
      5. Compile and upload the sketch. Monitor Serial at 115200 baud to verify sensor reads and Wi‑Fi connection.
      6. Open dashboard.html in a browser. Data tiles should update; use Quick Access to export logs as needed.

      Hardware Setup

      • Mount the ESP32 in a ventilated enclosure, away from water splashes.
      • Use a regulated 5V, 2A supply for sensors and relays; share ground with the ESP32.
      • Keep analog sensor wires short and twisted to reduce noise. Use waterproof probes and strain relief near tanks.
      • Relays must be rated above the pump/feeder load; use flyback diodes for DC loads and suitable snubbers for AC loads.

      Wiring Guide

      Reference mapping (adjust pins if your firmware uses different assignments):

      Temperature (DS18B20)

      Data → GPIO 4, Vcc 3.3V/5V, GND. Add a 4.7kΩ pull‑up between Data and Vcc.

      pH Sensor (Analog)

      Signal → GPIO 34 (ADC1), Vcc 5V (per board), GND. Use shielded cable.

      Dissolved Oxygen (Analog)

      Signal → GPIO 35 (ADC1), Vcc per module, GND. Calibrate in air/zero solution.

      Turbidity (Analog)

      Signal → GPIO 32 (ADC1), Vcc 5V, GND. Keep probe vertical and avoid bubbles.

      Relays

      • Relay 1 — Feeder: IN → GPIO 14, Vcc 5V, GND. Use separate PSU for motors if possible.
      • Relay 2 — Drain Pump: IN → GPIO 27, Vcc 5V, GND. Add breaker/RC snubber for AC pumps.
      • Relay 3 — Refill Valve/Pump: IN → GPIO 26, Vcc 5V, GND. Float switch in series recommended.

      I2C (if used)

      SDA → GPIO 21, SCL → GPIO 22, Vcc, GND. Add 4.7kΩ pull‑ups.

      First Run Checklist

      • Power the ESP32 and confirm it does not overheat and the onboard LED blinks.
      • Open the Serial Monitor: verify Wi‑Fi connection and that an IP address is shown.
      • Open the Dashboard: tiles should show values within expected ranges and no critical alerts.
      • Trigger actuators briefly to confirm relay operation and observe load behavior.
      • Calibrate sensors following TURBIDITY_CALIBRATION_GUIDE.md and related documents.

      Troubleshooting

      Wi‑Fi not connecting
      1. Verify SSID and password are correct in the firmware.
      2. Confirm the ESP32 is in range and the network supports 2.4GHz (ESP32 often requires 2.4GHz).
      3. Check router logs for blocked devices or MAC filtering.
      Sensors show 0 or erratic values
      1. Verify wiring, shared ground, and sensor power supply voltages.
      2. Keep analog cables short and use shielded cable; add filtering if necessary.
      3. Ensure correct ADC pins are used and that the firmware config matches the wiring.
      Relays click but load doesn’t run
      1. Confirm the relay contacts are rated for the load and wiring is correct.
      2. For motors, ensure sufficient inrush current capacity or use a dedicated motor driver.
      Data not appearing on dashboard
      1. Check the Firebase configuration and ensure the firmware is writing to the same database path used by the dashboard.
      2. Open the browser console for errors and verify real‑time listeners are attached.

      Maintenance

      • Clean probes weekly in distilled water; recalibrate monthly.
      • Inspect relay terminals for corrosion and tighten screws.
      • Backup data exports (CSV/PDF) monthly.

      Safety Notes

      • Keep mains wiring isolated; use proper enclosures and strain relief.
      • Add GFCI/RCD protection in wet environments. Never handle live wires near water.
      • Label all connectors and disconnect power before servicing.
